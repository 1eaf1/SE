## Лабораторная работа №2
### 2.2 Настройка
![Меню](https://github.com/1eaf1/SE/blob/Тема_1/pic/2.2%20Настройка.png)
## Выводы
В данном номере происходит настройка/авторизация в Git при помощи команд ```git config --global user.name "имя пользователя"``` и ```git config --global user.email "почта пользователя"```. Проверка установки/настроек осущетсвляется с помощью команды ```git --list```. Помимо авторизации в git можно настроить редактор для сообщений при помощи команды ```git config --global core.editor "code --wait"``` - в данном случае происходит смена редактора Vim на Visual Studio Code.
